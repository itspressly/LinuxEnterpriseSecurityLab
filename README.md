# Linux Enterprise Security Lab

## Overview

This project demonstrates the deployment and administration of a secure Ubuntu Linux server in a virtual machine.

The project simulates a real-world enterprise environment where a Linux System Administrator is responsible for configuring users, managing permissions, hosting services, securing the operating system, automating administration tasks, and documenting the infrastructure.
